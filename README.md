## Getting Started

Create Virtual Environment
```
pyenv virtualenv 3.9.5 aoc
pyenv activate aoc

pip install pip --upgrade
pip install -r requirements.txt
```

Install pre-commit hooks:
```
pre-commit install
```

Running tests:
```
pytest 2022
```
